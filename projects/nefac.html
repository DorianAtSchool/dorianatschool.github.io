AI-powered legal research for First Amendment rights.

About This Project

NEFAC (New England First Amendment Coalition) is a multi-agent RAG system developed in collaboration with the New England First Amendment Coalition. The platform helps journalists, researchers, and citizens navigate First Amendment law through an intelligent question-answering system.

Key Features

  • Multi-Agent Architecture – Specialized agents for different aspects of legal research
  • RAG System – Retrieval-Augmented Generation for accurate legal information
  • Production-Ready – Deployed and serving real users
  • AWS Infrastructure – Scalable and secure cloud deployment

Technical Stack

  • LangChain – LLM orchestration and agent framework
  • FastAPI – Backend API framework
  • React – Frontend interface
  • AWS – Cloud infrastructure
  • Python – Primary backend language

Team

As Project Lead at Build UMass, I led a team of 13 software developers and project managers on this initiative with the New England First Amendment Coalition, delivering a production-ready system for legal research assistance.
