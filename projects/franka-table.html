About This Project

A simulation environment of 4 Franka Panda arms on a table for multi-agent RL training with synthetic data generation, using MuJoCo and Gymnasium.

README

This project creates a simulation environment for training multi-agent reinforcement learning policies on robotic manipulation tasks.

Features

  • 4 Franka Panda Arms - Simulated robotic arms arranged around a table
  • MuJoCo Physics - High-fidelity physics simulation
  • Gymnasium Interface - Standard RL environment API
  • Synthetic Data Generation - Generate training data for various tasks

Use Cases

  • Multi-agent coordination learning
  • Collaborative manipulation tasks
  • Synthetic dataset generation for imitation learning

Technical Stack

  • MuJoCo - Physics simulation engine
  • Gymnasium - RL environment interface
  • Python - Primary programming language
